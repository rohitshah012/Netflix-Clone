# 🎬 Netflix Clone

A modern and responsive **Netflix landing page clone** built using HTML and CSS.
This project replicates the UI of Netflix with a clean layout, responsive design, and smooth user experience.

🔗 **Live Demo:**
https://rohitshah012.github.io/Netflix-Clone/

---

## 🚀 Features

* 🎨 Clean and modern Netflix UI
* 📱 Responsive design (mobile, tablet, desktop)
* 🧭 Interactive navigation bar
* 🖼️ Hero section with background banner
* 🎞️ Multiple content sections (movies/shows layout)
* ⚡ Lightweight and fast

---

## 🛠️ Tech Stack

* 🌐 HTML5
* 🎨 CSS3 (Flexbox & Grid)

## 📸 Preview

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/rohitshanh012/Netflix-Clone.git
```

2. Open the project folder:

```bash
cd Netflix-Clone
```

3. Open `index.html` in your browser.

---

## 🌐 Deployment

This project is deployed using **GitHub Pages**.

If the site shows as **"Dangerous"**, it may be due to:

* ⚠️ Mixed content (HTTP images instead of HTTPS)
* ⚠️ External resources not secured
* ⚠️ Browser security warnings

👉 **Fix Suggestions:**

* Use only `https://` links for images and assets
* Avoid suspicious external links
* Check browser console for errors
